# ⭐ ETNAir — Plateforme de location à court et moyen terme

Bienvenue sur le dépôt officiel du projet **ETNAir**, une application prototype développée dans le cadre d’un hackathon entre jeunes entrepreneurs de l’ETNA.  
L’objectif : **réinventer la location courte et moyenne durée** en proposant une plateforme moderne, scalable et simple d’utilisation.

## 🧠 Contexte

ETNAir est une startup fictive fondée lors d’un hackathon.  
Notre équipe a **4 semaines** pour concevoir un prototype fonctionnel, comprenant :

- Une API Node.js  
- Un front-end moderne  
- Une base de données PostgreSQL  
- Une infrastructure Docker/Kubernetes

---

## 🎯 Objectifs du prototype

### 👤 Côté utilisateur
- Les **locataires** peuvent rechercher et réserver un logement.
- Les **propriétaires** peuvent publier et gérer leurs annonces.

### 🛠️ Côté administration *(à développer ultérieurement)*
- Tableau de bord des utilisateurs  
- Suivi des réservations  

### ⚙️ Côté technique
- API REST Node.js / Express  
- Base PostgreSQL  
- Conteneurisation via Docker  
- Environnement scalable avec Kubernetes  

---

## 👥 Rôles et collaboration

- **Développeurs** : API + Front-end  
- **Administrateurs système** : infrastructure, sécurité, pipelines
